# OpenVideo 设计基准（当前版）

## 基准图（当前）

当前项目 UI 对齐基准为以下三张：

- `design/全量界面.png`
- `design/竖屏播放界面.png`
- `design/横屏播放界面.png`
- `design/播放的设置页面.png`（设置页专用）

未出现在这些图里的页面和细节，不应在文档中写成“已定义设计”。

## 本版范围

本版对齐范围：

- 主列表页（`全量界面.png` 右上手机）
- 竖屏播放页（`竖屏播放界面.png`）
- 横屏播放页（`横屏播放界面.png`）
- 设置页（`播放的设置页面.png`）

## 设置页必须遵循的结构（`播放的设置页面.png`）

1. 左侧导航栏（固定）
   - 标题：播放器设置
   - 分类：播放 / 画面 / 声音 / 字幕 / 手势 / 其他
   - 底部入口：恢复默认设置
2. 右侧内容区（当前选中分类内容）
   - 分组卡片样式
   - 行级设置项（左侧标题，右侧值或开关）
   - 行间弱分隔线

## 播放分类（图中明确）

### 分组一：播放控制

- 播放速度：`0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x`
- 循环播放：关闭 / 单集循环 / 列表循环
- 跳过片头片尾：开关
- 快进/快退间隔：`5s / 10s / 15s`
- 记住播放进度：开关（附说明文案）

### 分组二：播放选项

- 播放时启用硬件加速：开关
- 退出时自动暂停：开关
- 自动连播下一集：开关
- 背景音（后台播放）：开关（附说明文案）

## 风格约束（按图）

- 深色底（蓝黑调），面板有层级，不使用亮色大面积背景
- 选中导航项必须有蓝色高亮块
- 主文字亮、次文字灰蓝，分隔线弱化
- 右侧行项目是“设置列表”而不是表单堆叠
